# Editorial baseline failure cases

Observed weak-output patterns before the editorial policy layer:

- childish drift when asked for more joy
- adjective inflation instead of stronger verbs
- over-editing lines that were already strong
- generic AI polish that blurs register
- insufficient respect for the selected span

Representative example:
- Input: `This is a magic unicorn, isn't it?`
- Weak output: `This is a sparkling little unicorn, isn't it?`
- Failure class: childish drift / decorative rewrite / low editorial taste
